    leap.emigration module¶

    class leap.emigration.Emigration(starting_year: int = 2000, province: str = 'CA', population_growth_type: str = 'LG', table: DataFrameGroupBy | None = None)[source]¶

    Bases: object

    A class containing information about emigration from Canada.

    property table : DataFrameGroupBy¶

    Grouped dataframe (by year) giving the probability of emigration for a given age, province, sex, and growth scenario:

    • year: integer year the range 2001 - 2065.

    • age: integer age.

    • M: the probability of a male emigrating.

    • F: the probability of a female emigrating.

    See processed_data/migration/emigration_table.csv.

    load_emigration_table(starting_year: int, province: str, population_growth_type: str) → DataFrameGroupBy[source]¶

    Load the data from processed_data/migration/emigration_table.csv.

    Parameters:¶
    starting_year: int¶

    the year for the data to start at. Must be between 2001-2065.

    province: str¶

    a string indicating the province abbreviation, e.g. “BC”. For all of Canada, set province to “CA”.

    population_growth_type: str¶

    Population growth type, one of:

    • past: historical data

    • LG: low-growth projection

    • HG: high-growth projection

    • M1: medium-growth 1 projection

    • M2: medium-growth 2 projection

    • M3: medium-growth 3 projection

    • M4: medium-growth 4 projection

    • M5: medium-growth 5 projection

    • M6: medium-growth 6 projection

    • FA: fast-aging projection

    • SA: slow-aging projection

    See: StatCan Projection Scenarios.

    Returns:¶

    A dataframe grouped by year, giving the probability of emigration for a given age, province, sex, and growth scenario.

    compute_probability(year: int, age: int, sex: str | Sex) → bool[source]¶

    Determine the probability of emigration of an agent (person) in a given year.

    Parameters:¶
    year: int¶

    The calendar year, e.g. 2022.

    age: int¶

    Age of the person.

    sex: str | Sex¶

    Sex of the person, “M” = male, “F” = female.

    Returns:¶

    True if the person emigrates, False otherwise.

    Examples

    >>> emigration = Emigration()
>>> emigration.compute_probability(year=2022, age=0, sex="F")
False
