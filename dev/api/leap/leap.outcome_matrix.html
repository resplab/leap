    combine(outcome_table: leap.outcome_matrix.OutcomeTable, columns: list[str])[source]¶

    Combine two outcome tables via summation.

    This method combines the current outcome table with another OutcomeTable instance by summing the values in the specified columns. The tables must have the same structure and groupings.

    Parameters:¶
    outcome_table: leap.outcome_matrix.OutcomeTable¶

    The outcome table to combine with this one.

    columns: list[str]¶

    The columns to combine. These should be the columns that contain the values to be summed. If the columns are not present in both tables, a ValueError will be raised.
