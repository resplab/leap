    property population_growth_type : str¶

    Population growth type to be used in the simulation.

    One of:

    • past: historical data

    • LG: low-growth projection

    • HG: high-growth projection

    • M1: medium-growth 1 projection

    • M2: medium-growth 2 projection

    • M3: medium-growth 3 projection

    • M4: medium-growth 4 projection

    • M5: medium-growth 5 projection

    • M6: medium-growth 6 projection

    • FA: fast-aging projection

    • SA: slow-aging projection

    See StatCan.

    get_num_new_agents(year: int, min_year: int, num_new_born: int, num_immigrants: int) → int[source]¶

    Get the number of new agents born/immigrated in a given year.

    For the first year, we generate the initial population. For subsequent years, we generate the sum of the number of newborn babies and the number of immigrants to Canada.

    Parameters:¶
    year: int¶

    The calendar year of the current iteration, e.g. 2027.

    min_year: int¶

    The calendar year of the initial iteration, e.g. 2010.

    num_new_born: int¶

    The number of babies born in the specified year.

    num_immigrants: int¶

    The number of immigrants who moved to Canada in the specified year.

    Returns:¶

    The total number of new agents born/immigrated in the specified year.

    get_new_agents(year: int) → pandas.core.frame.DataFrame[source]¶

    Get the new agents born/immigrated in a given year.

    Parameters:¶
    year: int¶

    The calendar year.

    Returns:¶

    A dataframe containing a list of new agents to add to the model.

    The dataframe has the following columns:

    • age: The age of the agent.

    • sex: The sex of the agent.

    • immigrant: Whether or not the agent is an immigrant. If the agent is not an immigrant, they are a newborn, except if it is the first year of the model. In that case, the agent is part of the initial population.

    Examples

    >>> from leap.simulation import Simulation
>>> from leap.utils import get_data_path
>>> config_path = get_data_path("processed_data/config.json")
>>> simulation = Simulation(config=config_path, min_year=2027, num_births_initial=5)
>>> new_agents_df = simulation.get_new_agents(year=2028)
>>> list(new_agents_df["immigrant"]) 
[True, True, True, True, True, True, False, False, False, False, False]

    simulate_agent(sex: str, age: int, is_immigrant: bool, year: int, year_index: int, month: int) → tuple[UUID4, OutcomeMatrix][source]¶

    Simulate a new agent in the model.

    The agent in this function is a person who is either:

    1. A new immigrant to Canada in the specified year.

    2. A newborn baby born in the specified year.

    This function simulates the agent’s life from birth/immigration until death or emigration.

    Parameters:¶
    sex: str¶

    One of "M" or "F".

    age: int¶

    The age of the agent.

    is_immigrant: bool¶

    Whether or not the agent (person) is an immigrant.

    year: int¶

    The current year of the simulation.

    year_index: int¶

    The index of the current year in the simulation. For example, if the simulation starts in 2010, then the year index for 2010 is 0, for 2011 is 1, etc.

    month: int¶

    The month of the year when the agent is born/immigrates.
