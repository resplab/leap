    leap.utils.compute_ordinal_regression(η: float, θ: float | list[float], prob_function: typing.Callable = <function sigmoid>) → list[float][source]¶

    Compute the probability that y = k for each value of k.

    The probability is given by ordinal regression:

    \[\begin{split}P(y <= k) &= σ(θ_k - η) \\ P(y == k) &= P(y <= k) - P(y < k + 1) \\ &= σ(θ_k - η) - σ(θ_{(k+1)} - η)\end{split}\]
    Parameters:¶
    η

    The weight for the regression.

    θ

    Either a single value or an array of values for the threshold parameter.

    prob_function

    A function to apply; default is the sigmoid function.

    Returns:¶

    A vector with the probability of each value of k.

    For example:

    k=1 | k=2  | k=2
0.2 | 0.75 | 0.05

    Examples

    >>> compute_ordinal_regression(0, 0.5)
[np.float64(0.6224593312018546), np.float64(0.3775406687981454)]
>>> compute_ordinal_regression(0, [0.5, 1.5])
[np.float64(0.6224593312018546), np.float64(0.19511514499178906), np.float64(0.18242552380635635)]
